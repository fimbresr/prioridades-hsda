# Prioridades · HSDA

PWA de gestión de prioridades con **matriz de Eisenhower** para el Hospital San Diego de Alcalá.

- Captura tareas (con duración estimada y fecha límite opcional).
- Clasifícalas arrastrándolas (o tocándolas) a uno de los 4 cuadrantes: **Hacer ya · Agendar · Delegar · Eliminar**.
- **Mi día** arma automáticamente la agenda por prioridad, encadenando bloques según la duración de cada tarea.
- Funciona offline, instalable como app. Los datos se guardan solo en el dispositivo.

Construida con React (CDN) en un solo `index.html`. Sin backend.
